Project 3 Personalization Phase: Bridging the gap between raw user data and relevant content through pure similarity logic.

📌 Overview
A simple content-based recommendation system that matches user preferences with item attributes using Cosine Similarity. Users provide their interest levels for different categories, and the algorithm ranks items in the database by calculating vector similarity, displaying match percentages with visual indicators.

✨ Key Features
Takes user input (preferences rated 0-10)
Matches preferences using Cosine Similarity algorithm
Displays ranked recommended items with visual match percentage bars
Pure logic-based pattern matching (no machine learning libraries required for matching)

🛠️ Tech Stack
Language: Python 3.x
Libraries: NumPy
Concepts: Content-Based Filtering, Vector Similarity, Recommendation Systems
